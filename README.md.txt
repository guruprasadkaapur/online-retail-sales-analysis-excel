# Online Retail Sales Analysis Dashboard (Excel)

## Project Overview

This project analyzes online retail transaction data and creates an interactive Excel dashboard to understand sales performance and customer behavior.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Excel Formulas
- Data Cleaning
- Data Visualization

## Dataset

Online Retail Transaction Dataset

Columns:
- InvoiceNo
- StockCode
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country

## Data Cleaning

Performed:
- Removed missing customer records
- Removed cancelled transactions
- Removed invalid quantities and prices
- Created Revenue column

## Analysis Performed

- Total Revenue Analysis
- Monthly Sales Trend
- Country-wise Sales Performance
- Top Products by Revenue
- Customer Revenue Analysis

## Dashboard

Created KPI cards and charts:

- Total Revenue
- Total Orders
- Total Customers
- Average Order Value
- Sales Trend
- Product Performance

## Key Skills

Excel | Data Cleaning | Pivot Tables | Dashboard | Business Analysis
